# Titanic Exploratory Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python.

## Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

## Project Tasks
- Data Cleaning
- Handling Missing Values
- Feature Engineering (Age Group & Family Size)
- Survival Analysis by Age Group
- Survival Analysis by Embarkation Port
- Survival Analysis by Family Size
- Age Distribution Visualization
- Correlation Heatmap

## Dataset
Titanic Dataset (train.csv)
